# Ultra-Low Latency RTSP Media Player

This project is an ultra-low latency RTSP (Real-Time Streaming Protocol) media player specifically designed for drones, CCTV (Closed-Circuit Television), and IP cameras. Our player aims to provide a seamless streaming experience with minimal delay, ensuring you can monitor and react to live feeds in real time.

## Features
- **Ultra-Low Latency**: Optimized for fast streaming and low delay.
- **Support for Multiple Protocols**: Compatible with a variety of RTSP streams from different vendors.
- **User-Friendly Interface**: Simple and intuitive UI for quick access to streaming.
- **Lightweight Design**: Minimal resource consumption to operate efficiently on drones and edge devices.

## Applications
- **Drones**: Stream essential video feeds in real time for surveillance and reconnaissance missions.
- **CCTV**: Enhance surveillance capabilities with live monitoring.
- **IP Cameras**: Provide immediate access to video streams, enhancing security and operational monitoring.

## Getting Started
To use this media player, simply connect it to your RTSP camera's feed, and you will gain access to live streaming with ultra-low latency.
